Harmonic Trends

Musicians remember harmonic phrases and reuse them in new settings. This project turns chord-sequence data into interactive views of that vocabulary: what is common, what is genre-specific, and how phrase length changes the picture.

About The Project

These modules are static exports from the Harmonic Trends analysis pipeline. The underlying project cleans raw chord data, builds exact and harmonic n-gram vocabularies, stores the outputs in DuckDB, and produces trend, genre, embedding, and inference-ready tables.
