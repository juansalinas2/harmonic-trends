Harmonic Trends

Musicians remember harmonic phrases, the contexts where they work, and the continuations that feel natural. This Space turns chord-sequence data into interactive views of that vocabulary: what is common, what is genre-specific, how phrase length changes the picture, and what tends to come next.

About The Project

These modules are static exports from the Harmonic Trends analysis pipeline. The underlying project cleans raw chord data, builds exact and harmonic n-gram vocabularies, stores the outputs in DuckDB, and produces trend, genre, embedding, and inference-ready tables.
