H8 harmonic embedding: genre signatures vs global frequent loops

Hover or focus any point to inspect the full harmonic n-gram. Gray points are common H8 classes, the dark points are globally frequent loops, and colored points are genre-distinctive examples selected by lift.

This interactive view uses the same processed PCA and enrichment outputs as the static README figure. PCA axes are descriptive coordinates, not direct musical dimensions; the useful signal is the neighborhood structure and the contrast between corpus-wide frequency and genre-specific lift.
